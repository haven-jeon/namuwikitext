# Namuwikitext

Wikitext format Korean corpus

Corpus size
- train: 38278040 lines (5.3G)
- dev: 197723 lines (28M)
- test: 193614 lines (29M)

To fetch data, run below script. Then three corpus, train / dev / test files are downloaded at `./data/`

```
python fetch.py
```
